Java에서 화살표(`->`)는 람다 표현식(lambda expression)에서 사용되는 문법입니다. 람다 표현식은 함수형 프로그래밍 개념을 지원하기 위해 Java 8부터 도입된 기능입니다.

람다 표현식은 일반적으로 함수형 인터페이스(Functional Interface)를 구현하기 위해 사용됩니다. 함수형 인터페이스는 하나의 추상 메서드를 가지는 인터페이스를 말하며, 이를 통해 람다 표현식을 사용할 수 있습니다.

람다 표현식의 기본 문법은 다음과 같습니다:

```java
(parameter) -> { body }
```

- `parameter`: 람다 함수의 매개변수를 나타냅니다. 매개변수가 없을 경우 괄호를 비워둘 수 있습니다. 여러 개의 매개변수가 있는 경우 쉼표로 구분합니다.
- `->`: 화살표 연산자로, 매개변수와 람다 함수의 구현부를 구분합니다.
- `{ body }`: 람다 함수의 구현부를 나타냅니다. 중괄호 안에 원하는 코드를 작성합니다.

예를 들어, 두 수를 더하는 간단한 람다 함수를 작성해보겠습니다:

```java
// 인터페이스 선언
interface Calculator {
    int add(int a, int b);
}

// 람다 표현식 사용
Calculator calculator = (a, b) -> a + b;

// 람다 함수 호출
int result = calculator.add(2, 3);  // 결과: 5
```

위 예시에서 `Calculator`는 함수형 인터페이스로, `add`라는 하나의 추상 메서드를 가지고 있습니다. 이를 람다 표현식을 사용하여 구현하고, `calculator` 변수에 할당하였습니다. 그리고 `add` 메서드를 호출하면 람다 함수가 실행되어 두 수를 더한 결과를 반환합니다.

람다 표현식은 간결하고 함수형 프로그래밍 스타일의 코드 작성을 지원합니다. 이를 통해 코드의 가독성과 효율성을 높일 수 있습니다.

Subtractor subtractor = (a, b) -> a - b;

// 람다 함수 호출
